# Sketchpad

Raw, unfiltered capture for random thoughts, half-formed ideas, and "what if" notes.
Paste freely here — nothing in this file is committed, planned, or load-bearing.

**Workflow:** dump here first → refine later → graduate the keepers into [[phases]]
(or an ADR in `decisions/` if it's an architectural choice). Delete or strike through
ideas that don't survive.

**Agents:** treat everything here as tentative. Do not implement, cite, or act on a
sketchpad note unless the user explicitly promotes it. This is not project memory —
it is a holding pen.
